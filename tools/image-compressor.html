Frequently Asked Questions

Which image formats can I compress?

LovelyDoc supports all popular web formats including **JPEG/JPG, PNG, and WebP**. We are constantly adding support for more formats based on user feedback.

Is there a file size limit?

Because the tool runs entirely on your device, the only limit is your device's memory. We've successfully compressed images as large as 50MB without any issues.

Does compression reduce quality?

Any "lossy" compression (like JPEG) technically reduces data, but our tool uses optimized settings so that **you won't notice any difference** with your naked eye at recommended levels.

Why are my files never uploaded?

We use modern browser technology (Web Workers and Canvas) to process your data locally. This is faster and 100% more secure than traditional online tools that require you to upload private files to their servers.
